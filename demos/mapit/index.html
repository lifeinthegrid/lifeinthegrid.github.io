Welcome to MapIt

A jQuery plugin powered by Google Maps.

Overview

Simplified Clean Customizable

MapIt Pro is a jQuery plug-in that interacts with the Google Maps API. MapIt groups and organize all your locations into an easy to use interface. All the heavy lifting is done so you don't have mess with learning the Google API calls. Simpliy edit a basic XML file and your ready to share your locations.

MapIt uses a layout philosophy centered on stylesheets. Since it is impossible to know the different variations in which this plug-in could be used, we have implemented several CSS selectors in order for you to customize MapIt. We strongly encourage users to use the Style Locator Tool with a DOM inspector to locate the style you want to change.
